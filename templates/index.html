Data Dashboard

Interactive Data Visualization with Flask and D3

Author: Thanh Khang Nguyen

About the Project: This is a short demo for Interactive charts for data visualisation using Flask and D3.

Summary of Project Steps:

  • Setup: Install Flask, create basic structure.
  • Data Handling: Prepare and serve data via Flask routes.
  • D3.js Integration: Include D3.js for interactive visualizations.
  • Chart Creation: Use D3.js to create SVG-based interactive charts.
  • Integration: Connect Flask backend with D3.js frontend.
  • Styling: Apply CSS for aesthetics and responsiveness.

About Dataset: Student Performance in Exam

  • Marks secured by the students in high school who from the United States
  • This data set consists of the marks secured by the students in various subjects.
  • Source: Kaggle

Student Demographics: Gender and Race Distribution

We have two charts:
  • Pie Chart: Shows the percentage of male and female students.
  • Bar Chart: Shows the distribution of students' race.
When no section of the pie chart is clicked, the bar chart shows the overall race distribution. When a section of the pie chart is clicked, the bar chart updates to show the race distribution for the selected gender.
